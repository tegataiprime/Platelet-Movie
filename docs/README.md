# GitHub Pages Site

This directory contains the static website for Platelet-Movie, hosted via GitHub Pages.

## Files

- **index.html** - Main HTML page structure
- **styles.css** - Styles with CSS variables for light/dark mode theming
- **app.js** - JavaScript for:
  - Theme toggle and persistence
  - Data loading from data.json
  - Sortable table columns
  - Dynamic rendering
- **data.json** - Movie data with Lady Whistledown commentary (auto-generated by GitHub Actions)

## Features

### Responsive Design
The site is fully responsive and works on:
- Desktop browsers (Chrome, Firefox, Safari, Edge)
- Tablets (iPad, Android tablets)
- Mobile devices (iOS, Android)

### Light/Dark Mode
Users can toggle between light and dark themes. The preference is stored in localStorage and persists across visits.

### Sortable Table
Click any column header to sort the movie list:
- **Runtime** - Sort by movie length
- **Score** - Sort by TMDB rating
- **Rated** - Sort by MPAA certification
- **Genres** - Sort alphabetically by primary genre
- **Title** - Sort alphabetically by movie title

## Auto-Generated Content

The `data.json` file is automatically generated every Friday at 8:00 PM UTC by the `deploy-github-pages.yml` workflow.

**Do not manually edit data.json** - it will be overwritten by the next automated deployment.

## Local Development

To test the site locally:

```bash
# Using Python's built-in HTTP server
cd docs
python3 -m http.server 8000

# Or using Node.js http-server
npx http-server docs -p 8000
```

Then visit http://localhost:8000 in your browser.

## Manual Data Generation

To manually generate data.json for testing:

```bash
# Generate movies JSON
poetry run platelet-movie --format json > movies.json

# Generate commentary
poetry run platelet-movie --format markdown | poetry run python scripts/lady_whistledown.py > commentary.txt

# Build data.json (use the build script from the workflow)
python build_data.py
```
